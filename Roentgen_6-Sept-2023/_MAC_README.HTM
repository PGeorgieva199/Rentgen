Viewing medical images on Mac OS

This disc contains confidential medical information. If you are not the intended recipient or an authorized user, please contact the site shown on this label.

This disc contains DICOM images (in the \DICOM folder) and/or other medical information.

DICOM Viewer software is required to view DICOM images, and might need to be installed by an Administrator.

Many DICOM Viewers are available for Mac OS, including:

  • Miele-LXIV
  • Horos
  • Osirix
